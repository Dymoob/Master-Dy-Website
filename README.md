## 📌 About This Project

This repository documents the ongoing frontend work behind the official **Master Dy** website — a real, live production environment that has been continuously maintained, updated, and improved over **6+ years**.

This is not a course project or a mockup. Every change made here has gone live on a real website with real visitors.
